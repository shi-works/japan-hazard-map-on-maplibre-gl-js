## 国土地理院
- 各種ハザードマップ（PNGタイル形式）

    - 概要：「重ねるハザードマップ」に掲載しているデータをURLからリアルタイムに読み込み、ウェブサイトやソフトウェア、アプリケーションに商用非商用問わず利用することが可能。
